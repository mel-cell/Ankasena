# Ankasena Website

This is a monorepo containing the frontend and backend for the Ankasena project.

## Structure

- **frontend**: Nuxt.js application
- **backend**: PayloadCMS (Next.js) application

## Development

This project uses NPM Workspaces.

### Prerequisites

- Node.js (v20+ recommended)
- npm (v10+)

### Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run development servers:
   ```bash
   npm run dev
   ```

## Scripts

- `npm run dev`: Starts both frontend and backend in development mode.
- `npm run build`: Builds both applications.
- `npm run test`: Runs tests for both applications.
- `npm run lint`: Lints both applications.
